# TmSDK

This project is a SDK for controlling TMC80/160/256 cameras.

## Development

Windows C++
- Windows 11
- Visual Studio 2022
- Qt5.14.2
- qtcreator

Windows C#
- Windows 11
- Visual Studio 2022

Windows Python
- Windows 11
- Visual Studio 2022
- Python 3.9 or higher
- PyQt5
- qtcreator

Linux C++
- Ubuntu 24.04
- Gcc-11
- Qt5.14.2
- qtcreator

Linux Python
- Ubuntu 24.04
- Python 3.9 or higher
- PyQt5
- qtcreator
